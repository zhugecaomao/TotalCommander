-------------------------------------------------------------------------------
   kByte
-------------------------------------------------------------------------------
kByte - контентный плагин WDX для Total Commander.
Плагин позволяет получать несколько первых, средних или последних килобайт 
данных из файла и выводить их в виде HEX или MD5 строки.

ВНИМАНИЕ! Неаккуратное сравнение файла! Если совпали выбранные килобайты, 
то файлы считаются одинаковыми! Убедитесь, что Вы понимаете, что делаете!

-------------------------------------------------------------------------------
  История изменений
-------------------------------------------------------------------------------
Версия 0.2
- в полях MD5 появилась возможность выбрать объем сравниваемых данных 1, 4, 16, 64, 256 килобайта
- добавлено поле дынных first+middle+last
Версия 0.1
- вывод в HEX и MD5 первого, последнего и среднего килобайта данных
- сравнение содержимого файлов по первому, последнему и среднему килобайту

-------------------------------------------------------------------------------
Список полей:
first - данные из начала файла
middle - данные из "средних килобайтов" файла
last - данные из конца файла
first+middle+last - все вышеперечисленные данные
  HEX - hex-строка данных
  MD5 - md5-хеш данных
  *1k - количество килобайтов для считывания

"средние килобайты" (middle) получаются равномерно из всего файла
[first|    |middle|    |middle|    |middle|    |last]
[<-- начало файла                    конец файла -->]
